## ðŸ› ï¸ Local Setup

1. **Clone the repo:**
   ```bash
   git clone https://github.com/vivekbarnaon/micro-wins-ai.git
   cd micro-wins-ai
   ```
2. **Backend:**
   ```bash
   cd backend
   pip install -r requirements.txt
   func start
   ```
3. **Frontend:**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
4. **Open in browser:**
   - Frontend: [http://localhost:5173](http://localhost:5173)
   - Backend: [http://localhost:7071/api](http://localhost:7071/api)
